#  Risk Entropy Analyzer

A lightweight quantitative analytics tool that measures **portfolio volatility** and **information entropy** to assess diversification efficiency.

###  Features
- Computes stock and portfolio volatility  
- Calculates Shannon entropy for diversification insight  
- Generates automated visual risk reports  
- 2 faster reporting compared to manual spreadsheet analysis  

### 锔 Tech Stack
Python 路 Pandas 路 NumPy 路 Matplotlib 路 Yahoo Finance API

###  Results
Achieved **2 faster** report generation and delivered **entropy-based diversification metrics** for simulated 4-stock portfolios.
